# Добро пожаловать

Этот учебник собирается автоматически из Markdown файлов, размещённых в репозитории Gitea. Чтобы добавить новую статью:

1. Создайте файл `docs/articles/имя-статьи.md`.
2. Добавьте его в секцию `nav` файла `mkdocs.yml`.
3. Добавьте изображения в `docs/assets/images` и используйте относительные пути.
4. Закоммитьте и запушьте изменения.

После пуша Git hook запустит сборку MkDocs и обновит статический сайт, который раздаёт Nginx.
