# SA Knowledge Hub

Библиотека для аналитиков и продакт-команд: тёмная Material-тема, быстрая навигация, практические статьи и чек-листы. Готовый продукт — заходи, читай, применяй.

## Что внутри
- Три уровня сложности: **Новичок → Специалист → Эксперт**.
- Гайды по требованиям, исследованиям, моделированию процессов, архитектуре, интеграциям.
- Конспекты по C4, SWOT/PESTEL, User Story/BRD/MRD, CJM, приоритизации, рискам.
- Кастомные элементы темы: вкладки, прогресс, чек-листы.
