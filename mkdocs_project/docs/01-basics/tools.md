# Инструменты (Beginner)

Чтобы уверенно стартовать, соберите набор базовых инструментов.

## Документация и заметки

- Confluence или Notion для структурирования знаний.  
- Google Docs / Office Online для оперативного совместного редактирования.

## Коммуникации

- Slack/Telegram/Teams — основной канал общения с командой.  
- Miro/Mural — визуализация идей, customer journey и быстрые воркшопы.

## Моделирование

- Draw.io, Diagrams.net или PlantUML для простых схем.  
- Bizagi Modeler / Camunda Modeler для BPMN-диаграмм.

## Дополнительные помощники

- Figma — интерфейсы и пользовательские сценарии.  
- Postman — проверка API.  
- VS Code + Markdown расширения — ведение технических заметок.
