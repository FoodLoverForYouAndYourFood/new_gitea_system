# Новичок: инструменты

Набор, чтобы быстро стартовать и не утонуть в выборе.

## Документация и заметки
- Markdown (VS Code / Obsidian) — лёгкий формат, ссылки и чек-листы.
- Confluence / Notion — когда нужен совместный доступ и версии.
- Google Docs — быстрые черновики, комментарии.

## Воркшопы и схемы
- Miro / FigJam — доски, коллаборативные схемы.
- Draw.io — простые диаграммы без аккаунта.

## Хранение требований и задач
- Jira / YouTrack — задачи, спринты, статусы.
- Trello / Linear — для небольших команд и pet-проектов.

## Исследования и аналитика
- Google Analytics / Яндекс.Метрика — продуктовые метрики.
- SQL/BI (Metabase, Redash, Superset) — быстрые отчёты.
- Hotjar/Session replay — понять, как пользователь кликает.

## Коммуникации
- Zoom/Meet/Teams — созвоны и записи интервью.
- Slack/Telegram/Discord — обратная связь и каналы поддержки.

## Полезные шаблоны
- Одностраничник фичи (one-pager) — цель, метрика, сценарий, критерии.
- Глоссарий — роли, объекты, события.
- Чек-лист интервью — вопросы «зачем/что/когда/как измеряем».
