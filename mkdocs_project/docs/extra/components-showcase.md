# Демонстрация компонентов

Пара примеров, которые можно использовать в своих заметках.

## Вкладки

```tabs
=== API
- Метод: `POST /api/v1/orders`
- Авторизация: Bearer Token
- Response time: `< 200 ms`

=== UI
- Сценарий: Пользователь оформляет заказ
- Основные проверки: наличие товара, лимиты оплаты
- Edge cases: отложенная доставка, предоплата
```

## Admonition

!!! tip "Лайфхак"
    Используйте admonition, чтобы выделить важные детали, ссылки на стандарты или подсказки для новичков.

!!! warning "Риски"
    Регулярно проверяйте актуальность инструкций — устаревший контент может навредить больше, чем его отсутствие.

## Прогресс-бар

```html
<div class="progress-wrapper">
  <div id="progress-bar" class="progress-bar" style="width: 0%"></div>
</div>
<p>Прогресс изучения: <span id="progress-value">0</span>%</p>
<button class="sa-mark-read" data-article="components-demo">Отметить материал пройденным</button>
```

Прогресс хранится в `localStorage`, поэтому обновления видны только вам — идеальный вариант для личного трекера.
