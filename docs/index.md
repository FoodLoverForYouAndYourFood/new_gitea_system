# SA Knowledge Hub

Добро пожаловать в **SA Knowledge Hub** — компактную библиотеку системного аналитика в тёмной фиолетовой тематике Material for MkDocs.

- Тёмный стиль Dark Pro с фиолетовыми акцентами  
- Структура по уровням роста: Beginner, Middle, Pro  
- Демонстрация компонентов и шаблонов  
- Система прогресса на JavaScript с хранением в LocalStorage

> Заглядывайте в разделы по уровням зрелости и отмечайте изученные материалы — прогресс обновляется мгновенно благодаря instant view.
